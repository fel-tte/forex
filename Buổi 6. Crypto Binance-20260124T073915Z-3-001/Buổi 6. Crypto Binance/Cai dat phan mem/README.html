Cài đặt phần mềm

Yêu cầu hệ thống

  • Python 3.8 trở lên
  • pip (Python package manager)

Các bước cài đặt

  1. Cài đặt Python
  2. Cài đặt các thư viện cần thiết
  3. Cấu hình môi trường

Hướng dẫn chi tiết

1. Cài đặt Python

  • Tải Python từ trang chủ: https://www.python.org/downloads/
  • Chọn phiên bản Python 3.8 hoặc cao hơn
  • Trong quá trình cài đặt, đảm bảo tích chọn "Add Python to PATH"

2. Cài đặt các thư viện

pip install -r requirements.txt

3. Cấu hình môi trường

  • Tạo file .env trong thư mục gốc của dự án
  • Cấu hình các biến môi trường cần thiết

Kiểm tra cài đặt

python --version
pip --version

Hỗ trợ

Nếu gặp vấn đề trong quá trình cài đặt, vui lòng liên hệ hỗ trợ.
