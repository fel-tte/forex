# Hướng dẫn tạo tài khoản các sàn giao dịch

## 1. Hướng dẫn tạo tài khoản Binance

### Bước 1: Đăng ký tài khoản
- Click vào đường dẫn: https://www.binance.com/en/activity/referral-entry/CPA?ref=CPA_00I0P0XG68
- Nhập thông tin đăng ký bằng số điện thoại hoặc email

### Bước 2: Xác thực tài khoản
- Làm theo hướng dẫn chi tiết tại: https://www.binance.com/vi/support/faq/h%C6%B0%E1%BB%9Bng-d%C3%A0nh-cho-ng%C6%B0%E1%BB%9Di-m%E1%BB%9Bi-b%E1%BA%AFt-%C4%91%E1%BA%A7u-s%E1%BB%AD-d%E1%BB%A5ng-binance-c780097f75dd450a82d17f1e84153276

## 2. Hướng dẫn tạo tài khoản Forex

### Bước 1: Cài đặt DNS
- Cần cài đặt DNS trước khi đăng ký

### Bước 2: Đăng ký tài khoản
- Click vào đường dẫn: https://my.xs.com/vi/links/go/971
- Làm theo hướng dẫn đăng ký trên trang web

## 3. Hướng dẫn tạo tài khoản Chứng khoán

### Bước 1: Đăng ký tài khoản
- Truy cập đường dẫn: https://www.ssi.com.vn/khach-hang-ca-nhan/mo-tai-khoan?moigioi=1463&mgm=I580035
- Điền đầy đủ thông tin theo yêu cầu

### Bước 2: Đăng ký API
- Sau khi có tài khoản, tiến hành đăng ký API để có thể giao dịch tự động

## Lưu ý quan trọng
- Bảo mật thông tin tài khoản
- Không chia sẻ API key và Secret key
- Lưu trữ thông tin đăng nhập an toàn
- Thực hiện xác thực 2 lớp (2FA) nếu có
